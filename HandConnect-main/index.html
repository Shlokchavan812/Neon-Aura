🎮 Complete Gesture Guide

⚡ Basic Gestures

Pinch → Thumb + Index touch
Fist → All fingers closed
Open Hand → All fingers extended
Thumbs Up → Thumb pointing up
Thumbs Down → Thumb pointing down
OK Sign → Thumb+Index circle

🎮 Rock-Paper-Scissors

Rock → Closed fist
Paper → All fingers extended
Scissors → Index+Middle spread

🎉 Celebratory Gestures

Victory/Peace → V-shape with 2 fingers
Three Fingers → Index+Middle+Ring
Four Fingers → All 4 fingers extended

🖖 Advanced Shapes

Spock → Star Trek V-shape
Love/Heart → Thumb+Pinky+Index extended
Claw → All fingers wide spread
Holy Sign → Index+Middle wide apart
Gun → Thumb+Index only

☝️ Pointing Gestures

Point Up → Index only upward
Point Right → Index only rightward

🤝 Dual-Hand Gestures

Hands Touching → Palms very close
Hands Close → Hands near each other
Hands Crossing → Hands overlapping

💡 Tips

  • ✅ Use clear, exaggerated gestures for best results
  • ✅ Ensure hands are fully visible in camera frame
  • ✅ Good lighting improves hand detection accuracy
  • ✅ Gestures have a 300ms cooldown to prevent re-triggering
  • ✅ Watch the HUD for real-time gesture feedback
  • ✅ Particles and haptics confirm gesture recognition
  • ✅ Two-hand gestures work when both hands are detected
